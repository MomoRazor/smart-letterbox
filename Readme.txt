The aim of this project is to be a very simple smart letterbox. By smart, it is simply meaning that the letter box knows that it has something inside of it. However, this modules must:

- Be easy to install into existing letterboxes (not a letterbox within itself)
- Be completely self powered
- Be able to remotely notify users of its status
- Be able to remotely notify users of its current power levels

Therefore on a technical level, this module have:

- Power storage
- Power monitoring
- Network connecting capabilities

Off module, the system must include:

- A server to receive Network connections
- An app to communicate with server
